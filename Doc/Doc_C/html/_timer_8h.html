Detailed Description

Timer module header.

This file defines the interface used to manage 64-bit timer. This module use XTmrCtr library and is based on xtmrctr_polled_example.c.

Initializes AXI timer IP registers.

The timer is initialized in cascaded mode in order to obtain a 64-bit timer. The period of such a timer @100MHz is more than 5845 years before wrap.

Parameters:
baseAddressis the register base address of the AXI timer IP.
clockFreqis the timer clock frequency or the ticks count per second.
