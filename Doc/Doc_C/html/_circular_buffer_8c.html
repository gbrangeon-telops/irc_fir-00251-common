Return the byte at the tail of the circular buffer.

This function returns the byte at the tail position of the circular buffer. The tail position is NOT updated so the byte has to be popped or flushed later.

Parameters:
circBufferis the pointer to the circular buffer structure.
indexis the index of the byte to be returned relative to the tail (from 0 to circular buffer length -1).

Returns:

IRC_FAILURE if the byte at index is not available.
