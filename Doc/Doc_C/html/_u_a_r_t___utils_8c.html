Detailed Description

UART utility module implementation.

This file implements the interface used to manage UART link. This module use XUartNs550 library based on xuartns550_intr_example.c.

Configures UART link.

Encapsulates XUartNs550_SetDataFormat function in order to configure the UART link.

Parameters:
uartis the pointer to the UART link data structure.
baudRateis the baud rate of the UART device in bps, ie 1200.
dataBitsis the number of data bits of the UART device.
parityis the parity of the UART device ('N' = None, 'O' = Odd and 'E' = Even).
stopBitsis the number of stop bits of the UART device.
Returns:
IRC_SUCCESS if the data format was successfully set.
IRC_FAILURE if an error occurred while setting the UART link format (XST_UART_BAUD_ERROR or XST_INVALID_PARAM).

Initializes UART link.

Initializes UART link by initializing the UART device data structure. It also connects and enables UART interrupt.

Parameters:
uartis the pointer to the UART link data structure.
uartDeviceIdis the UART device ID that can be found in xparameters.h file.
intcis the pointer to the Interrupt controller instance.
uartIntrIdis the UART interrupt ID that can be found in xparameters.h file.
rxBufferis a pointer to the received data buffer.
rxBufferSizeis the size of received data buffer.
txBufferis a pointer to the sent data buffer.
txBufferSizeis the size of sent data buffer.
handleris a the pointer to the UART interrupt handling function (Must complies to XUartNs550_Handler function prototype).
callbackRefis a pointer to the data that is passed to the handler function as reference.
Returns:
IRC_SUCCESS if successfully initialized
IRC_FAILURE if failed to initialize.
